# SRL Dashboard (Streamlit v3)

**What's new**
- Phases in **three tabs** (Forethought, Performance, Reflection)
- **Refresh visuals** button (uses `st.rerun()`)
- Global chat limiter (15 interactions/hour) with seeded per-phase conversations

## Quickstart
```bash
pip install -r requirements.txt
streamlit run app.py
```
